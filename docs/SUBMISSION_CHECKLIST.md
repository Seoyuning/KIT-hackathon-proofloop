## 제출 전 확인

### 기본 동작
- [ ] 저장소가 `Public`인지 확인
- [ ] `.env`, API 키, 토큰이 커밋되지 않았는지 확인 (`.env.example`만 있어야 함)
- [ ] 라이브 URL이 열리는지 확인
- [ ] `/` 접속 시 스플래시 → `/studio/login`으로 리다이렉트 되는지 확인

### 회원가입/로그인
- [ ] 학생 회원가입: 이메일, 비밀번호, 이름, 역할(학생), 학년(중1~고3) 선택 가능
- [ ] 교사 회원가입: 이메일, 비밀번호, 이름, 역할(교사) 선택 가능
- [ ] 테스트 계정으로 로그인 정상 동작

### 교사 플로우
- [ ] 처음 로그인 시 대시보드에 하드코딩 데이터 없이 빈 상태
- [ ] 반 관리에서 반 만들기 (학년/과목/출판사/교과서 입력)
- [ ] 초대 코드 클릭 시 클립보드 복사 + "복사됨!" 피드백
- [ ] 사이드바에서 반 선택 시 운영 상태/교과서 정보 전환
- [ ] 여러 반 관리 가능 (반 간 전환)
- [ ] 질문 분석 대시보드: 누적 질문, 참여 학생, 단원별 이해도 표시
- [ ] 시드 데이터에 "(예시)" 태그 표시
- [ ] 수업 자료/시험 초안 생성 동작
- [ ] 내보내기 (클립보드 복사, 파일 다운로드)
- [ ] 학습 자료 추가 (사진 촬영/선택, 파일 업로드)

### 학생 플로우
- [ ] 반 미참여 시 챗봇 입력 차단 + "반에 먼저 참여해 주세요" 안내
- [ ] 초대 코드 입력 후 반 참여 성공
- [ ] 반 참여 후 챗봇 활성화 (해당 반의 교과서 정보로 전환)
- [ ] 질문 전송 → 교과서 근거 카드 + 이해도 바 + 후속 질문 카드 표시
- [ ] "이 문제 풀어줘" 입력 → AI가 답 거절 + 서술형 질문 발행
- [ ] 마이페이지에서 학년 정보 표시

### 교사-학생 연결
- [ ] 학생이 질문한 후, 교사 대시보드에서 누적 질문 수 증가 확인
- [ ] 반 관리에서 해당 학생의 약점 개념/최근 질문 확인

### 보안
- [ ] 학생이 교사 페이지 접근 시 리다이렉트
- [ ] 교사가 학생 페이지 접근 시 리다이렉트
- [ ] 비로그인 상태에서 API 호출 시 401 반환
